# BMIcalculatorMobileApp
# 📱 BMI Calculator

A simple and intuitive **Body Mass Index (BMI) Calculator** Android application built with Android Studio.

---

## 📖 Description

BMI Calculator is a lightweight Android app that helps users quickly calculate their Body Mass Index based on their height and weight. The app provides instant results along with a health classification to help users understand their BMI range.

---

## ✨ Features

- ⚖️ Calculate BMI using height and weight inputs
- 📊 Display BMI result with health category (Underweight / Normal / Overweight / Obese)
- 🔄 Support for both **metric** (kg/cm) and **imperial** (lbs/ft) units
- 🎨 Clean and user-friendly Material Design UI
- ⚡ Instant real-time calculation

---

## 🛠️ Built With

- **Language:** Java / Kotlin
- **IDE:** Android Studio
- **Min SDK:** 21 (Android 5.0 Lollipop)
- **Target SDK:** 34 (Android 14)

## 🚀 Getting Started

### Prerequisites

- Android Studio (latest version recommended)
- Android SDK installed
- A physical device or emulator running Android 5.0+

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/BMIcalculatorMobileApp.git
   ```

2. Open the project in **Android Studio**:
   - Go to `File` → `Open` → Select the project folder

3. Sync Gradle:
   - Click **"Sync Now"** when prompted

4. Run the app:
   - Connect your Android device or start an emulator
   - Click the ▶️ **Run** button or press `Shift + F10`

## 📐 BMI Categories

| BMI Range | Category |
|-----------|----------|
| Below 18.5 | Underweight |
| 18.5 – 24.9 | Normal weight |
| 25.0 – 29.9 | Overweight |
| 30.0 and above | Obese |
